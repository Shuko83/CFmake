Replace Functions

append(variablename, suffix)

Appends each value stored in variablename with suffix.

For example:

LIBRARIES = MyLib1 MyLib2
LIBRARIES = $$append( LIBRARY, .dll )   # LIBRARIES = MyLib1.dll MyLib2.dll

See also prepend().

prepend(variablename, prefix)

Prepends each value stored in variablename with prefix.

For example:

LIBRARIES = MyLib1.so MyLib2.so
LIBRARIES = $$prepend( LIBRARIES, lib )   # LIBRARIES = libMyLib1.so libMyLib2.so

See also append().

remove(variablename, substr)

Removes each instance that matches the regular expression substr in the contents of variablename.

For example:

MESSAGE = This is a test example.
MESSAGE = $$remove( MESSAGE, [ ]?example )
message( $${MESSAGE} )   # prints: "This is a test."

TARGET([mode])

Returns the contents of TARGET according to the current build mode. If TARGET does not contain any debug modifier, this function returns the same value regardless of the given mode.

mode can have one of the following values: debug or release (fallback value if argument is not specified or invalid).

For example:

CONFIG += debug_and_release
TARGET = MyApp[d]

message( $$TARGET() )            # prints: "MyApp"
message( $$TARGET( release ) )   # prints: "MyApp"
message( $$TARGET( debug ) )     # prints: "MyAppd"

See also Target Definition.
