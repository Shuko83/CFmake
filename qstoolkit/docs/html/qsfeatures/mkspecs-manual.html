qmake Features

Overview

This set of features extends default qmake features, in order to simplify and unify the generation of development projects across different platforms and build environments. The following table lists configurations supported by these features.

Qt 4.x

PlatformCompilerqmake Platform and Compilers Scope
Desktop Platforms
Windows 7
(32 and 64-bit)		MSVC 2010, MSVC 2008
MinGW 4.4		plaftorm: win32
compiler: msvc, msvc-proj, mingw, gcc
Linux / X11
(32 and 64-bit)		GCC 4.2 and aboveplaftorm: linux, gnu
compiler: gcc

Qt 5.x

PlatformCompilerqmake Platform and Compilers Scope

(32 and 64-bit)		MSVC 2015, MSVC 2013, MSVC 2012, MSVC 2010
MinGW 4.7 and above		plaftorm: win32
compiler: msvc, msvc-proj, mingw, gcc
Linux / X11
(32 and 64-bit)		GCC 4.8 and aboveplaftorm: linux, gnu
compiler: gcc
Mobile Platforms
Android
(x86, armv5, armv7)		GCC 4.8 and above (provided by Android NDK)
Toolchain: Windows MinGW		plaftorm: mobile, android, linux
compiler: gcc

Note: QS features requires qmake 2.01a (Qt 4.7) or later to be processed.

Installation

QS features directory location is free. To be taken into account by qmake tool, just set QMAKEPATH environment variable to qstoolkit directory path.
